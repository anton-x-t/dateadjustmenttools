Install the plugin "Date adjustment tools"

Prerequisites: in order to install this plugin, adminstrative priviligies are needed.

1. Unzip dateadjustmenttools-master.zip and rename it "dateadjustmenttools".

2. Move the folder "dateadjustmenttools" to the folder "wwwroot/moodle/local/" (wwwroot is the LAMP/WAMP/XAMPP main server folder, e.g. C:\Moodle\server\moodle\local\).

3. Go to "wwwroot/admin/index.php" in your browser (e.g. <http://localhost/admin/index.php>).

4. Click on "Check for available updates".

5. If the plugin was added correctly the server should now prompt for installation, click "Upgrade Moodle database now".

6. After some time the page should load and say "Success". Click "Continue" on that page.

7. Go to a course home page on your Moodle site.

8. Look under "Course administration", there should now probably exist a node below "Question bank" named "Date adjustment tools".

9. Click on the link "Date adjustment tools" to use the plugin.
