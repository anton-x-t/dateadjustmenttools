Install the plugin "Date adjustment tools"

Prerequisites: in order to install this plugin, adminstrative priviligies are needed.

1. Unzip dateadjustmenttools-master.zip and rename it "dateadjustmenttools".

2. Move the folder "dateadjustmenttools" to the folder "wwwroot/moodle/local/" (wwwroot is your LAMP/XAMPP/WAMP main server folder, e.g. C:\Moodle\server\moodle\local\).

3. Go to "localhost/admin/index.php" in your browser. Click for available updates if Moodle doesn't recognize the plugin.

4. If the plugin was added correctly the server should now prompt for installation, click "Upgrade Moodle database now".

5. After some time the page should load and say "Success". Click "Continue" on that page.

6. Go to a course home page on your Moodle site.

7. Look under "Course administration", there should now probably exist a node below "Question bank" named "Date adjustment tools".

8. Click the link "Date adjustment tools" to use the plugin.
